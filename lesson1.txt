1.	Компоненты веб-разработки:
o	HTML (HyperText Markup Language): Основа, задаёт структуру (заголовки, списки, изображения).
o	CSS (Cascading Style Sheets): Стили, отвечают за внешний вид.
o	JavaScript: Интерактивность, добавляет динамику.
o	Серверная часть (кратко): Хранение данных, API.

2.	Как работает веб?
o	Браузер запрашивает страницу у сервера.
o	Сервер отправляет HTML, CSS, JavaScript.
o	Браузер рендерит страницу.
o	Инструмент: Chrome DevTools (F12) для анализа и отладки.

3.	Инструменты разработчика:
o	VS Code: Редактор кода с расширениями.
o	Live Server: Расширение для автоматического обновления страницы.
o	Chrome: Браузер для тестирования.
o	Папка проекта: web-course для всех файлов.

Основы HTML
HTML — язык разметки, использующий теги для описания структуры.
1.	Теги и элементы:
o	Тег: <tag>Содержимое</tag> (например, <p>Текст</p>).
o	Самозакрывающиеся: <img src="image.jpg" />.
o	Элемент: Тег + содержимое + атрибуты.

2.	Основные теги:
o	<h1>–<h6>: Заголовки.
o	<p>: Параграф.
o	<ul>, <ol>, <li>: Списки (ненумерованный, нумерованный).
o	<img>: Изображение (src, alt).
o	<a>: Ссылка (href).
o	<div>: Контейнер.
o	<nav>, <section>, <article>: Семантические теги.

3.	Структура HTML-документа:
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
</head>
<body>
<!-- Контент -->
</body>
</html>
<!DOCTYPE html>: Версия HTML5.
<html lang="ru">: Язык документа.
<head>: Метаданные.
<meta name="viewport">: Адаптивность.
<body>: Видимый контент.

4.	Семантика и доступность:
o	Семантические теги: <header>, <main>, <footer>, <nav>, <article> улучшают читаемость и SEO.
o	Атрибуты доступности:
alt для <img> (описание для скринридеров).
aria-label для интерактивных элементов.
role для указания роли (например, role="navigation").

5.	Атрибуты:
o	id: Уникальный идентификатор.
o	class: Группировка.
o	data-*: Пользовательские данные.
o	Пример: <a id="home-link" class="nav-link" aria-label="На главную">.
